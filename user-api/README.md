# MSA User API 
MSA 강좌 중 예제 User API 소스코드입니다.

# Getting Started
1. 로컬 컴퓨터에서 docker-compose 로 mysql 설치
```shell
docker-compose up --build
```

3. Application 실행 및 API 접속 테스트 
```shell
$ open http://localhost:8084/api/users/1
```
# Swagger 
```shell
open http://localhost:8084/swagger-ui/index.html
```
